float ( trôi )
        float: left; trôi sang trái
        float: right: trôi sang phải
        float: none; không trôi là giá trị mặc định
        ///////
        khi thẻ con bị float thì thẻ cha trực tiếp không chứa được thẻ con
        nếu thẻ cha không có thuộc tính height
        => fix lỗi này
        thêm class có tên là
        .clearfix::after {
            content: "";
            display: block;
            clear: both;
        }
        thêm class .clearfix cho thẻ cha
